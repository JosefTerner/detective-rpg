# Detective RPG Frontend

This directory contains the frontend application for the Detective RPG game.

## Structure

- **detective-rpg-ui/** - The main React application for the game UI

## Overview

The frontend provides an immersive interface for players to engage with the Detective RPG game. It communicates with the backend services through the API Gateway and provides a rich, interactive user experience for solving mysteries and navigating the game world.

## Getting Started

See the README.md in the detective-rpg-ui directory for specific instructions on how to set up and run the UI application.
